자유롭게 글을 남겨주세요

학생, 교수, 동문, 방문객 누구나 의학교육실에 전하고 싶은 의견·후기·격려를 남길 수 있습니다.

🔐
GitHub 계정으로 로그인

스팸 방지와 책임감 있는 소통을 위해 GitHub 계정 로그인을 사용합니다. 계정이 없다면 무료로 가입할 수 있습니다.

💬
마크다운 지원

볼드·이탤릭·링크·코드 등 마크다운 문법을 사용할 수 있고, 이모지(👍❤️🎉)도 사용 가능합니다.

🌐
공개 게시판

작성한 내용은 사이트 방문자 모두에게 공개됩니다. 정중하고 건설적인 의견을 부탁드립니다.

⚙️ 관리자 안내

Giscus 위젯을 활성화하려면 GitHub 저장소에 Giscus 앱 설치 후 giscus.app에서 발급받은 data-repo-id / data-category-id를 위 코드에 채워넣으면 됩니다. 설치 가이드는 README를 확인하세요.
