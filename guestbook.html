⚙️ 관리자 — 방명록 활성화 절차

본 사이트의 방명록은 GitHub Discussions 기반 Giscus로 동작합니다. 다음 4단계를 차례로 진행하면 위 자리표시자가 실제 댓글창으로 자동 전환됩니다.

  1. 1단계 — Giscus 앱 설치

    github.com/apps/giscus 접속 → Install 클릭 → Only select repositories → cnumededu 선택

  2. 2단계 — Discussions 기능 활성화

    저장소 설정 → Features → Discussions 체크박스 On

  3. 3단계 — Giscus 구성 (ID 발급)

    giscus.app/ko 접속 → 저장소 입력란에 universetimer/cnumededu → 매핑은 특정 용어, 카테고리는 General 선택. 페이지 아래에 생성된 <script> 태그에서 data-repo-id 와 data-category-id 값을 복사

  4. 4단계 — 값 알려주기

    발급받은 두 ID를 알려주시면 guestbook.html에 반영해 즉시 활성화하겠습니다 (또는 README의 코드 스니펫 부분을 직접 수정해 git push 해도 됩니다).
