⚙️ 관리자 — 방명록 활성화 절차 (1회 설정)

방명록은 Cloudflare Pages Functions + KV로 동작합니다. Cloudflare 대시보드에서 KV 네임스페이스를 한 번만 연결하면 됩니다.

  1. 1단계 — KV 네임스페이스 생성

    Cloudflare 대시보드 → Workers & Pages → 좌측 KV → Create a namespace → 이름: cnumededu-guestbook 입력 후 생성

  2. 2단계 — Pages 프로젝트에 바인딩

    Workers & Pages → cnumededu 프로젝트 → Settings → Functions → KV namespace bindings → Add binding
    · Variable name: GUESTBOOK
    · KV namespace: cnumededu-guestbook 선택 → Save

  3. 3단계 — 재배포

    Deployments 탭 → 최신 배포 옆 Retry deployment. 약 30초 후 위 폼이 실작동합니다.

  4. (선택) 관리자 삭제 기능

    Settings → Environment variables → Production → ADMIN_TOKEN 변수에 임의 시크릿 추가. DELETE /api/guestbook?id=... 요청 시 X-Admin-Token 헤더로 인증.
